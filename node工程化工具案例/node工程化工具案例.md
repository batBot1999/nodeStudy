# package.json
配置bin相当于此电脑全局支持命令,对应一个文件去执行,相当于双击执行这个文件,但js执行需要环境支持,不然只能在node或浏览器环境执行,
 "#!/"这个可以让命令跨平台生效: #!/usr/bin/env node,从这个目录找到node执行,和scripts可以达到一样的效果,但script是执行脚本,
 出了这个项目文件夹就无法执行了,

```js
    {
        "bin": {
            "qq-run": "qq.exe", // 执行qq
            "node-cli": "index.js"
        }
    }
```
